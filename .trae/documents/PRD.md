# 富文本编辑器 PRD（仿 uEditor）

## 1. 产品概述

本项目是一款基于 TypeScript 的所见即所得（WYSIWYG）富文本编辑器，界面与核心功能仿照百度 UEditor 设计。目标是提供一套经典、直观、功能完备的 Web 端富文本编辑能力，适用于内容管理、博客撰写、后台编辑等场景，可作为独立组件嵌入任意 Web 项目。

- **目标用户**：需要在网页中进行富文本内容编辑的开发者与内容创作者。
- **核心价值**：以 UEditor 经典交互形态降低学习成本，同时用现代 TypeScript + 模块化架构提供可维护、可扩展的编辑器内核。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 使用方式 | 核心权限 |
|------|----------|----------|
| 编辑者 | 直接在页面中使用编辑器 | 全部编辑、格式化、插入、源码切换等能力 |
| 开发者 | 通过 API 嵌入与配置编辑器 | 自定义工具栏、插件、获取/设置内容 |

### 2.2 功能模块

整体为单页编辑器应用，由四大区域构成：

1. **工具栏（Toolbar）**：分组展示格式化、插入、操作类按钮与下拉菜单。
2. **编辑区（Editor Body）**：所见即所得的内容编辑区域。
3. **状态栏（Status Bar）**：字数统计、选区路径、缩放比例等信息。
4. **弹层与菜单（Dialogs & Menus）**：链接/图片/表格属性弹窗、右键菜单、颜色选择器、源码视图。

### 2.3 工具栏分组明细

| 分组 | 按钮/功能 | 说明 |
|------|-----------|------|
| 操作 | 撤销 / 重做 | 历史栈管理 |
| 文字样式 | 加粗 / 斜体 / 下划线 / 删除线 | 基础字形修饰 |
| 字体 | 字体家族 / 字号 | 下拉选择，含中文字体 |
| 颜色 | 文字颜色 / 背景色 | 调色板 + 最近使用色 |
| 段落 | 左对齐 / 居中 / 右对齐 / 两端对齐 | 段落对齐 |
| 列表 | 无序列表 / 有序列表 | 支持多级缩进 |
| 缩进 | 增加缩进 / 减少缩进 | 段落层级控制 |
| 行距 | 行距下拉 | 1.0 / 1.5 / 2.0 等 |
| 标题 | 段落格式下拉 | 正文 / H1-H6 / 引用 |
| 插入 | 超链接 / 图片 / 表格 / 分割线 / 特殊字符 / 表情 | 弹窗或直接插入 |

| 区域 | 模块 | 功能描述 |
|------|------|----------|

## 3. 核心流程

### 3.1 主要用户流

- **编辑流**：用户进入编辑器 → 工具栏选择格式 → 在编辑区输入内容 → 实时预览效果 → 状态栏更新统计 → 完成后通过 API 获取 HTML 内容。
- **插入图片流**：点击图片按钮 → 弹窗选择 URL 或上传 → 设置尺寸/对齐 → 插入到光标位置。
- **源码切换流**：点击源码 → 编辑区切换为 textarea 显示 HTML → 编辑源码 → 切回所见即所得，内容同步。
- **全屏流**：点击全屏 → 编辑器 fixed 铺满视口 → 再次点击还原。

## 4. 用户界面设计

### 4.1 设计风格

仿 UEditor 经典桌面端编辑器外观，克制、专业、信息密度适中：

- **主色调**：浅灰白底（`#ffffff` / `#f5f6f7`），工具栏 `#fafafa`，分隔线 `#e5e6eb`。
- **强调色**：UEditor 经典蓝 `#3b8cff` 用于激活态按钮、下拉高亮。
- **按钮风格**：扁平矩形图标按钮，hover 浅灰底 `#eef0f3`，激活态浅蓝底 `#e6f0ff` + 蓝字。
- **字体**：界面使用「PingFang SC / Microsoft YaHei / system-ui」；编辑区默认「宋体 / SimSun」正文风格，标题用黑体加重。
- **布局**：顶部工具栏（可多行折叠）+ 中部编辑区（带内边距、白底）+ 底部状态栏，整体带 1px 边框圆角容器。
- **图标风格**：线性 SVG 图标，统一 16px，与 UEditor 视觉一致。
- **弹窗风格**：居中模态，标题栏 + 表单 + 底部确认/取消按钮，UEditor 风格的紧凑表单。

### 4.2 区域设计概览

| 区域 | 模块 | UI 元素 |
|------|------|--------|
| 工具栏 | 操作组 | 扁平图标按钮、分组分隔竖线、hover tooltip |
| 工具栏 | 下拉组 | 触发按钮 + 浮层面板（字体列表/调色板/字号列表） |
| 编辑区 | 内容区 | 白底、内边距 16px、最小高度 300px、聚焦边框高亮 |
| 状态栏 | 统计区 | 左侧选区路径、右侧字数统计，灰底 `#fafafa` |
| 弹窗 | 表单 | 标题栏、输入框、单选、底部按钮组 |

### 4.3 响应式

- **桌面优先**：编辑器为桌面端工具，最小宽度 720px。
- **窄屏适配**：宽度 < 720px 时工具栏自动换行折叠，弹窗宽度自适应，编辑区最小高度降为 240px。
- **触摸优化**：按钮点击区域 ≥ 32px，下拉面板增大内边距以适配触控。

### 4.4 3D 场景

不适用。
